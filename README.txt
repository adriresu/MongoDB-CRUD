1- Install PRE-REQUIREMENTS
2- Execute this command on terminal to up MongoDB server:
    """     docker run --name mongo-CRUD -p 27017:27017 -d  mongo       """
    """     This is going to up a docker container server of mongoDB on local port: 27017   """
3- Execute this command to install Python requirements:
    """     pip install -r requirements.txt     """
4- Execute the program.
